###### Purpose:
Mobile application to display list of movie, its description and a sample trailer

Functionalities:
+ Display list of movies
+ Search movie
+ View movie details
+ Play movie trailer
+ Add/delete a movie

###### Tools and Techologies:
+ Java
+ Android Studio
